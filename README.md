# CleanWeb

掃除モードを有効化すると、ページ上でホバー中のDOM要素をクリックして掃除（非表示化）できるChrome拡張です。
掃除済み要素はURL単位で保存され、ページを再読み込みしても自動で掃除状態が再適用されます。
元に戻すモードでは掃除済み要素を再表示し、クリックで復活できます。

## インストール（開発モード）

1. Chromeで `chrome://extensions` を開く
2. 右上の「デベロッパーモード」をON
3. 「パッケージ化されていない拡張機能を読み込む」を選択
4. このディレクトリ（`CleanWeb`）を指定

## 使い方

1. 拡張アイコンをクリックしてポップアップを開く
2. 「掃除モードを開始」を押す
3. ページ上で掃除したい要素にマウスを合わせる（赤枠でハイライト）
4. クリックするとその要素が掃除状態（非表示）になる
5. 掃除モード中は画面全体に赤系オーバーレイと「掃除モードON」ラベルが表示される
6. 「元に戻すモードを開始」を押すと、掃除済み要素を一時表示する
7. 元に戻すモードでは掃除済み要素に薄緑オーバーレイが表示され、ホバー選択してクリックで復活できる
8. 元に戻すモード中の上部表示「元に戻すモード中」の下に、掃除済み要素を全復活するボタンが表示される
9. 停止は各モードの停止ボタンか `Esc`
10. 同じURLを再読み込みしたときも、復活していない掃除済み要素は自動で掃除状態になる

## ファイル構成

- `manifest.json`: 拡張設定（Manifest V3）
- `content.js`: 掃除/元に戻すモード本体（ホバー選択・掃除状態の永続化・復活）
- `popup.html`: ポップアップUI
- `popup.css`: ポップアップスタイル
- `popup.js`: 掃除モード/元に戻すモードのON/OFF制御
